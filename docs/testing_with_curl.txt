Testing with curl
=================

Authorize (v2.0)
----------------

    curl "http://su1.3scale.net/transactions/authorize.xml?provider_key=PROVIDER_KEY&app_id=APP_ID&app_key=APP_KEY"

Note: Don't forget the quotes around the url, otherwise bash interprets the ampersand character!


Authorize with predicted usage (v2.0)
-------------------------------------


    curl "http://su1.3scale.net/transactions/authorize.xml?provider_key=PROVIDER_KEY&app_id=APP_ID&app_key=APP_KEY&usage\[METRIC\]=1"

Note: The [ and ] have to be escaped as "\[" and "\]".

Authorize with predicted usage with multiple metrics (v2.0)

    curl "http://su1.3scale.net/transactions/authorize.xml?provider_key=PROVIDER_KEY&app_id=APP_ID&app_key=APP_KEY&usage\[hits\]=1&usage\[cputime\]=10"


Report (v2.0)
-------------

    curl -F provider_key=PROVIDER_KEY -F transactions[0][app_id]=APP_ID -F transactions[0][usage][hits]=1 http://su1.3scale.net/transactions.xml

Report with multiple transactions (v2.0)

    curl -F provider_key=PROVIDER_KEY -F transactions[0][app_id]=APP_ID0 -F transactions[0][usage][hits]=1 -F transactions[1][app_id]=APP_ID1 -F transactions[1][usage][hits]=1 http://su1.3scale.net/transactions.xml
